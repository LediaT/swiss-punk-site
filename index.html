What is Swiss Punk?

Swiss Punk emerged in the 1970s as a rebellious response to rigid Swiss typography.

Led by Wolfgang Weingart, this movement embraced chaos, intuition, and expression.

KEY CHARACTERISTICS

Typography

  • Mixed typefaces & weights
  • Diagonal and rotated text
  • Inconsistent letter spacing
  • Bold stair-stepped layouts

Layout

  • Breaking the grid system
  • Asymmetric compositions
  • Layered collage elements
  • Chaotic but intentional

Philosophy

  • Intuition = Analysis
  • Expression over perfection
  • Anti-corporate aesthetic
  • Push limits of legibility
